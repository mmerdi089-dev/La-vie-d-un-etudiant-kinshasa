# La vie d'un étudiant

Ce projet est un jeu narratif qui simule le quotidien d'un étudiant à Kinshasa.

## Description
Le jeu permet aux joueurs de découvrir les défis et les réalités de la vie estudiantine. Il met en avant les choix que les étudiants doivent faire tout au long de leur parcours académique.

## Fonctionnalités
- Narration basée sur les choix de l'utilisateur.
- Simulation de situations quotidiennes réelles à Kinshasa.
- Système de gestion de progression.

## Objectif
Le but de ce projet est de créer une expérience immersive qui reflète les expériences vécues dans les institutions comme l'ISTA, où je suis actuellement étudiant.
